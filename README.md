# Food Ordering Backend System

If you are Frontend Engineer, College Student currently learning Frontend Development using Reactjs, Angular or Vuejs. and you want Production Grade Free API that is already built, so that you can directly integrate it into your projects to get the real world experience how it works and how enterprises built their systems, 
Then this is the right place for you, you can add API into you project directly without having trouble into Free API hit limits or any charges.

## You can clone this Project in you local laptop or anywhere you want it to run or deploy to access the API.
##### Step 1:
```
git clone <URL of the Github Project>
```
##### Step 2:
```
cd backend-service
```
